# AWS를 이용한 주차관리시스템
<br/>

### 💡 프로젝트 소개
본 프로젝트는 '클라우드IoT서비스'에서 진행한 프로젝트입니다.

1) 초음파 센서를 이용하여 다가오는 차량을 감지하고 카메라 모듈을 이용하여 차량의 번호판을 촬영한 후 이미지로 S3 버킷에 저장

2) AWS Rekognition을 사용하여 촬영된 번호판 이미지의 텍스트 추출

3) 추출된 텍스트를 데이터베이스와 비교하여 같은 번호가 존재하면 출차, 없으면 입차 처리
